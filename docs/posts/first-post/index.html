关于Hugo

Hugo是一个用Go语言编写的静态网站生成器，它具有以下优点：

  • 速度快: 可以在几毫秒内生成整个网站
  • 灵活性强: 支持各种内容类型和布局
  • 易于部署: 生成的是静态HTML文件，可以托管在任何地方
  • 无需数据库: 所有内容都以Markdown文件形式存储

Stake主题特点

Stake是一个专为博客设计的Hugo主题，提供：

  • 🎯 简洁的设计
  • 📱 完全响应式
  • 🚀 快速加载
  • 💾 轻量级

开始使用

如果你想在你的本地机器上运行这个博客，可以执行以下命令：

hugo server -D

然后在浏览器中打开 http://localhost:1313
